# Edit Requests

Write your edit requests below. Each edit should specify:
- **Piece**: which stationery piece (save-the-date, invitation, details-card, rsvp-card, program, table-name-tag, food-menu, envelope-liner)
- **What**: what to change
- **Details**: specifics (new text, color, size, layout, etc.)

Changes are applied by Claude and visible instantly via live-server.

---

## How to use

1. Start the live preview server:
   ```
   npx live-server /Users/Max/wedding-stationery/ --open=preview/index.html
   ```
2. Write your edit requests below the `---` line
3. Tell Claude to apply them (or Claude will pick them up when you say "apply edits")
4. Changes appear in your browser immediately (auto-reload)
